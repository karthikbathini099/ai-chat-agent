# AI Chat Agent using n8n

This project demonstrates a simple AI chat agent built using n8n and Google Gemini.

## Features
- Chat trigger
- AI agent processing
- Google Gemini model
- Simple memory for conversation

## Architecture
User Message → AI Agent → Gemini Model → Memory → Response

## Tools Used
- n8n
- Google Gemini
- AI Agent Node
- Memory Node

## How to Use
1. Import the workflow JSON file into n8n
2. Add your Google Gemini API key
3. Run the workflow
4. Start chatting with the AI agent
